Site-web-Nvidia-responsive


Ce dossier comporte un fichier HTML5, un dossier CSS avec un fichier CSS3 ainsi qu'un dossier image comportant 10 images.
Le dossier comporte aussi un fichier PDF contenant une image du site web Nvidia à réaliser.

C'est une réplique (uniquement de la partie Front-end) d'un site de vente en ligne que l'on voit régulièrement sur le web.
Le thème est ici la marque : Nvidia

Le site répond aux nécessités du format responsive, il s'adapte à plusieurs formats :

- Moniteur/écran d'ordinateur : entre 860 et 1024px de largeur ou valeurs supérieurs
- Tablette : entre 480px et 860px de largeur
- Téléphone : 480px de largeur ou valeurs inférieurs
